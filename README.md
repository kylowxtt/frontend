# Atlas React Vite Website

High-fidelity React/Vite implementation of the generated Atlas landing page concept.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Structure

- `src/App.jsx` — section layout and component data.
- `src/styles.css` — full theme, desktop pixel-matched layout, and responsive behavior.
- `src/assets/` — Atlas reference imagery and extracted generated-image assets used by the page.

The desktop layout is tuned to the generated 1536 × 1024 reference image while still stacking into a mobile-friendly page below tablet widths.
