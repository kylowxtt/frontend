# Atlas Frontend

React/Vite implementation of the Atlas landing page concept: premium dark editorial styling, antique-gold and Atlas-blue accents, product workflow preview, proof modules, use cases, pricing, and final CTA.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

A GitHub Pages workflow is included at `.github/workflows/deploy.yml`. In GitHub, set Pages source to **GitHub Actions**.
